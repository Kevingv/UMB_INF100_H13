Velkommen til den nye INF100 siden!

Denne siden vil være informasjonssentralen for emnet INF100 denne høsten. Siden er plassert på GitHub, et av verdens største nettsider for lagring og deling av og samarbeid om kode, både i åpne og lukkede prosjekter.

Hvorfor GitHub?

Vi bruker GitHub i steden for Fronter til informasjonsdeling i INF100 siden GitHub er mye bedre egnet til deling av programmeringsorientert materiale. Dessuten er det (etter min mening) mye enklere i bruk enn Fronter. Vi skal allikevel benytte Fronter til innlevering av obligatoriske oppgaver, siden erfaringen fra tidligere kurs har vist at innlevering av materiale via Git kan være noe krevende, spesielt for begynnere.

Hva kan du gjøre på INF100s GitHub-side?

Hoveddelen av siden er arkivet (repository på engelsk), der du finner alle IPython Notebooks fra forelesningene, oppgavestillinger og eksempelløsninger til obligatoriske oppgaver samt tilleggsinformasjon som installasjonsveiledning for programvare og gruppeinndelingen til øvingene.

Du kan også stille spørsmål som berører kurset på Issues siden. Du må registrere deg på GitHub for å kunne stille spørsmål; registreringen er gratis. Er du registrert, kan du dessuten få automatiske varslinger hver gang noe nytt kommer i arkivet.

Hvis du har erfaring med Git fra før (eller vil lære deg det på egen hånd), kan du også klone hele arkivet direkte til egen maskin med

git clone https://github.com/heplesser/UMB_INF100_H13.git

IPython Notebooks og GitHub i undervisning: Et eksperiment

Så vel bruk av IPython Notebooks som interaktive forlesningsplansjer og av GitHub til utdeling av materiale to til diskusjon er et eksperiment. Jeg er meget interessert i deres tilbakemeldinger!
